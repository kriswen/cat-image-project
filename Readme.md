# # Cat Image API Project

Display random cat image in jpg or gif using public API. (https://cataas.com/)

## Authors

- [@kriswen](https://www.github.com/kris-wen)

## Deployment

To install dependencies

```bash
  npm install
```

To start this project run

```bash
  node index.js
```
